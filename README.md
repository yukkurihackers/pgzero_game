# pgzero_game
A simplified version of Space Invaders, used in a beginner tutorial about Python and Pygame Zero.

シンプルなスペースインベーダーゲームの改造版です。サウンドの差し替え、追加、敵キャラの変化など。


1. Clone or downloadでzipファイルをダウンロード。
2. 展開してフォルダに入る。
3. $ pgzrun game.py で起動。
